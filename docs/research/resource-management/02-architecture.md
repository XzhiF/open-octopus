### дёәд»Җд№Ҳ core еңЁ CLI иҖҢдёҚжҳҜ shared

PR #12 жҠҠж ёеҝғйҖ»иҫ‘еҲҶж•ЈеңЁ `shared/src/repository/` е’Ң `cli/src/repository/` дёӨеӨ„пјҢеҜјиҮҙеҸҢиҪЁй—®йўҳгҖӮ

**ж–°и®ҫи®Ў**пјҡ

| еұӮ | ж”ҫд»Җд№Ҳ | дёҚж”ҫд»Җд№Ҳ |
|---|---|---|
| `shared/src/resource/` | зәҜзұ»еһӢпјҲZod SchemaпјүгҖҒзәҜе·Ҙе…·еҮҪж•°пјҲhash/path/formatпјүгҖҒжҺҘеҸЈе®ҡд№ү | жңүзҠ¶жҖҒзҡ„ classгҖҒfs ж“ҚдҪң |
| `cli/src/resource/` | ResourceManager classгҖҒRegistryStoreгҖҒSourceProvidersгҖҒжүҖжңү fs ж“ҚдҪң | вҖ” |
| `server/src/services/resource/` | ResourceServiceпјҲи–„еҢ…иЈ…пјүвҶ’ и°ғз”Ё cli зҡ„ ResourceManager | йҮҚеӨҚе®һзҺ° |

Server йҖҡиҝҮ **import cli зҡ„ ResourceManager** еӨҚз”Ёж ёеҝғйҖ»иҫ‘пјҢиҖҢдёҚжҳҜиҮӘе·ұеҶҚе®һзҺ°дёҖйҒҚгҖӮ
